# Data Storage
: 수집한 디지털 데이터를 장기간 저장하고 효율적으로 관리하기 위한 저장소  

> 파일 시스템 > 데이터베이스 > 분산 저장 시스템 > 클라우드


**저장 장치**
- File System
- Database
- Data Warehouse
- Data Lake
- Data Mart

**저장 방식**
- Centralized
- Distributed
- Cloud
- Hybrid

**접근 방식**
- Block Storage
- Object Storage
- File Storage

**저장 방식**
- Centralized
- Distributed
- Cloud

**기술**
- RAID
- SAN
- NAS


**history**
```
1890년대 천공 카드 
1920년대 자기 드럼, 자기 테이프
1930년대 광학 데이터 저장 개념 제안 (에드윈 랜드)
1940년대 최초 1024 비트 정보 저장소 개발
1950년대 자기 디스크
1960년대 DRAM 셀
1970년대 버블 메모리, 최초 휴대용 저장 장치인 플로피 디스크 개발
1980년대 하드 디스크 대중화, CD-ROM
1990년대 플래시 메모리, DVD, NAS
2000년대 USB, SSD 대중화, 클라우드 스토리지 
```

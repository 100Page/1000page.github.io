# Hardware
: 컴퓨터를 구성하는 물리적 요소  

**구성 요소**
- [Motherboard](./motherboard.md)
- [CPU](./cpu.md)
- [Memory](./memory.md)
- [System Bus](./system-bus.md)
- [Electronic Circuits](./electronic-circuits.md)


```
데이터 -- 입력 장치 -- 시스템 버스 -- 출력 장치
                     중앙 처리 장치 (제어창지 + 연산장치)
                       기억 장치
                       저장 장치
```

**위치에 따라**
- 외부 장치 (키보드, 마우스 등)
- 내부 장치 (CPU, RAM 등)

**기능에 따라**
- 입력 장치 
- 제어 장치 : 명령어 해석 및 실행 순서 제어
- 처리 장치 : 데이터 처리 (연산 수행)
- 기억 장치 : 데이터와 명령어 임시 저장
- 저장 장치 : 데이터와 프로그램 영구 저장
- 출력 장치

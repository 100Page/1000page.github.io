# Circuits
: 회로란 전기가 흐를 수 있도록 구성된 경로나 시스템  

- 전기 회로 : 전류의 흐름을 다루는 모든 회로
  - 전자 회로
    - 아날로그 회로
    - 디지털 회로
- 광학 회로
- 유체 회로 



# Electronic Circuits
: 전자 회로란 전자 소자를 사용해 전기 신호를 생성하고 처리 및 전달하는 회로  
: 전기적 특성을 이용하는 오디오, 비디오, 데이터 통신 등에 사용됨  

**구성 요소**  
- 수동 소자 (저항, 커패시터 등)
- 능동 소자 (트랜지스터, 다이오드 등)
- 집적 회로 (IC)



## Digital Circuits
: 저항, 트랜지스터, 다이오드, 커패시터, 집적 회로 등으로 구성된 하드웨어  
: 디지털 컴퓨터를 구성하는 회로로 모든 연산과 데이터 처리는 디지털 회로를 통해 이루어짐  
: 디지털 회로는 전압(신호)을 0과 1의 상태로 나누어 처리함  

**기본 회로**
- [논리 회로](#logic-circuit)
- 타이밍 회로
- 인터페이스 회로



### Logic Circuit
: 논리 회로는 여러 [논리 게이트](#logic-gate)를 조합하여 구성된 회로로 논리 연산을 수행함  
: 전압을 1로 취급하는 정논리 회로나 전압을 0으로 취급하는 부논리 회로로 설계되며 일반적으로는 정논리가 사용됨  
: 저장 요소 유무에 따라 조합 논리 회로와 순차 논리 회로로 분류됨  

분류 | 설명 
---|---
[조합 논리 회로](#combinational-logic-circuit) | 저장 요소가 없는 회로로 입력에 따라 즉시 출력이 결정됨 
[순차 논리 회로](#sequential-logic-circuit) | 클럭 신호에 의해 동작하며 상태를 저장하고 현재 입력과 상태에 따라 출력이 결정됨



#### Logic Gate
: 논리 게이트는 [불대수 연산](../../mathematics/discrete-mathematics/boolean-algebra.md)을 물리적으로 구현한 회로로 한 개 이상의 입력 신호를 받아 하나의 출력 신호를 생성함  
: 논리 게이트는 기본적인 논리 연산을 수행하고 여러 논리 게이트가 결합되어 복잡한 논리 회로를 형성함  


**게이트**  
: 논리적으로는 불대수의 연산자를 의미하며 물리적으로는 이를 구현한 회로를 의미  
: 게이트는 트랜지스터, 저항 등 전자 소자들을 조합하여 구성되며 입력 신호를 처리해 출력 신호를 생성함  

게이트 | 설명
---|---
AND  | 두 개 이상 입력과 하나의 출력을 가지며 모든 입력이 1일 때 출력이 1인 논리 게이트
NAND | AND 게이트의 부정으로 입력이 하나라도 0이면 출력이 1인 논리 게이트 
OR   | 두 개 이상 입력과 하나의 출력을 가지며 입력이 하나라도 1이면 출력이 1인 논리 게이트
NOR  | OR 게이트의 부정으로 모든 입력이 0일 때 출력이 1인 논리 게이트
XOR  | 두 입력이 다를 때 출력이 1이고 두 입력이 같을 때 출력이 0인 논리 게이트
XNOR | XOR 게이트의 부정으로 두 입력이 다를 때 출력이 0이고 두 입력이 같을 때 출력이 1인 논리 게이트
NOT  | 하나의 입력과 하나의 출력을 가지며 입력이 0이면 출력이 1인 논리 게이트

NOT 게이트  
: 입력과 출력이 반대여서 인버트라고 표현하기도 함  


**논리 게이트 분류**

분류 | 게이트 (연산자) | 설명
---|---|---
기본 게이트 | AND, OR, NOT | 모든 논리 함수의 기본이 되는 게이트
유니버설 게이트 | NAND, NOR | 다른 모든 게이트를 구현할 수 있는 게이트
파생 게이트 | XOR, XNOR | 기본 게이트들의 조합으로 만들어진 게이트

**논리 게이트 완전 집합**  
: 논리 연산을 모두 구현할 수 있는 최소한의 논리 게이트 조합  
: NAND와 NOR 게이트만으로 모든 논리 함수를 구현할 수 있어 유니버설 게이트라고 함  



#### Combinational Logic Circuit
: 조합 논리 회로란 현재 입력에 따라 즉시 출력이 결정되는 논리 회로

```
입력 신호 => 조합 논리 회로 => 출력 신호
```

##### Adder 
: 덧셈을 수행하는 논리 회로  
: 덧셈을 수행할 때 합이 2 이상이면 올림수가 발생하며 올림수 처리 방식에 따라 반가산기와 전가산기로 분류됨  
: 구현 방식에 따라 병렬 가산기, 직렬 가산기, 고속 가산기로 분류됨  

분류 | 설명
---|---
반가산기 | 두 입력 비트를 받아 덧셈을 수행하고 합 비트와 올림수 비트를 출력 
전가산기 | 두 입력 비트와 한 개의 올림수 비트를 받아 덧셈을 수행하고 합 비트와 새 올림수 비트를 출력 
병렬 가산기 | 여러 비트를 동시에 계산하여 높은 속도를 제공
직렬 가산기 | 한 비트씩 클럭 신호에 따라 순차적으로 계산
고속 가산기 | 올림수 발생을 미리 예측하여 계산 속도를 향상함


**가감산기**  
: 덧셈과 뺄셈을 모두 수행하는 논리 회로  
: 가산기와 감산기 구조가 거의 동일해 가산기 회로에 제어 비트 회로를 추가하면 가감산기를 만들수 있음  



##### Comparator
: 두 입력 값을 비트 단위로 비교하여 크기 관계를 판단하는 논리 회로

**종류**
- 크기 비교기 : 두 입력 A와 B에 대해 세 가지 출력 A>B, A=B, A<\B 를 생성함
- 일치 비교기 : 두 입력이 동일한지 여부만을 판단



##### Encoder
: 부호화되지 않은 입력 신호를 받아 부호화된 출력 신호로 내보내는 논리 회로  
: 반대로 부호화된 신호를 원래 형태로 변환하는 회로를 디코더라고 함  
: 인코더를 통해 입력 신호를 간단한 이진 코드로 표현해 사용함  

- 인코더 = 부호화기
- 디코더 = 복호화기



##### Multiplexer
= MUX, 다중화기, 데이터 선택기  
: 멀티플렉서란 여러 입력 신호 중 하나를 선택하여 단일 출력으로 전달하는 논리 회로  
: 선택 신호를 통해 어떤 입력이 출력으로 연결될지 결정함  


**디멀티플렉서**  
: 하나의 입력 신호를 받아 여러 출력 중 하나로 전달하는 논리 회로  
: 1개의 입력을 n개의 선택 신호를 통해 2ⁿ개의 출력 중 하나로 활성화 함  



##### Code Converter
: 한 형태의 디지털 코드를 다른 형태의 코드로 변환하는 논리 회로  
: 특정 응용 분야에서 필요한 데이터 형식을 맞추기 위해 사용됨  



##### Parity Generator
: 데이터 전송 시 오류를 검출하기 위한 패리티 비트를 생성하고 검사하는 논리 회로  
: 패리티 생성기는 데이터 비트를 분석해 짝수 패리티나 홀수 패리티를 유지하기 위한 패리티 비트를 생성함   
: 패리티 검사기는 수신된 데이터의 패리티를 확인하여 오류를 검출함  

**패리티 비트**  
: 짝수나 홀수가 되도록 추가하는 비트  



#### Sequential Logic Circuit
: 순서 논리 회로란 상태를 저장하고 현재 입력과 저장된 상태에 따라 출력이 결정되는 논리 회로  
: 상태를 저장하기 위해 래치나 플립플롭같은 메모리 소자를 사용함  
: 클럭 신호로 제어되는 동기식 회로와 입력 변화에 따라 즉시 반응하는 비동기식 회로로 나뉨  

회로 | 설명
---|---
동기식 순서 논리 회로 | 클럭 신호에 동기화되어 상태 전이가 발생하는 회로 
비동기식 순서 논리 회로 | 클럭 신호 없이 입력 변화에 따라 즉시 상태가 전이되는 회로

# OpenSSF
Open Source Security Foundation  
: 오픈소스 생태계의 보안을 향상하기 위해 설립한 재단  

**목표**
- 오픈소스 프로젝트 보안 수준 향상
- 안전한 소프트웨어 개발과 배포 촉진
- 보안 도구 및 리소스 제공


**Initiative**
- Security Best Practices
- Training and Education
- Vulnerability Disclosure Guidelines
- Open Source Security Resources


**Project**
- SLSA
- Security Scorecards
- Dependency Review

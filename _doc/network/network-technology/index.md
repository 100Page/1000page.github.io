# Network Technology
: 네트워크를 구축하고 운영하기 위한 기술  


**Network Service**
- NAT
- DNS 
- DHCP
- QoS
- CDN 
- NTP 


**Network Transport**  
: 데이터 전송의 신뢰성과 효율성을 보장하는 네트워크 기술  

- STP
- BGP
- MPLS
- Multicast


**Network Virtualization**  
: 네트워크 리소스를 가상 환경에서 사용하도록 네트워크를 추상화한 논리적 네트워크 기술  

- [SDN](./sdn.md)
- [NFV](./nfv.md)


**Network Management**  
: 네트워크 성능과 상태를 모니터링 및 제어하는 기술  

- SNMP

# Communication Network
: 통신망이란 정보를 전송하기 위해 물리적 및 논리적 장치와 기술이 결합된 네트워크  
: 네트워크는 연결 구조를 포괄하는 광범위한 개념이며 통신망은 정보 전송이라는 목적을 가진 네트워크  

**전송 매체**
- 유선 통신
- 무선 통신
  - 이동 통신
  - 위성 통신

**통신 서비스**
- 음성통신망
  - PSTN
  - ISDN 
- 데이터통신망
  - PSDN
  - VAN

**기술**
- 다중화 기술
  - FDM
  - TDM
  - WDM
  - CDM
- 동기화 기술
  - 비트동기
  - 프레임동기
  - 네트워크동기

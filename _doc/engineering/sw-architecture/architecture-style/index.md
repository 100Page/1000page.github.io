# Architecture Style
: 소프트웨어 시스템의 전반적인 구조를 정의  
: 시스템 구성과 시스템 주요 구성 요소 간 상호작용에 대해 일관된 규칙과 지침을 제공  
: 시간이 지나며 특정 아키텍처 스타일이 정형화되면 아키텍처 패턴이 되기도 함

# JS Built-in Class
: 자바스크립트 엔진에 의해 제공되는 내장 클래스  
: 내장 클래스는 자바스크립트 엔진이 시작될 때 자동으로 메모리에 로드됨  
: 메모리에 로드되는 것은 클래스 정보이며 실제 객체는 인스턴스를 생성해야 메모리에 올라감  
: 내장 클래스는 new 키워드를 사용해 인스턴스를 생성해야만 사용 가능함

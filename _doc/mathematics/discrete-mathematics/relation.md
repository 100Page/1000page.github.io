# Relation
: 관계란 두 집합 간의 연관성 또는 대응을 나타내는 수학적 개념  
: 두 집합 X와 Y의 부분집합 R을 X에서 Y로의 관계라고 정의  
: 수학적으로 (x,y) ∈ R 또는 xRy 라고 표현함

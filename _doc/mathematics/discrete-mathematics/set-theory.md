# Set Theory
: 집합이란 순서가 없고 중복되지 않는 원소들의 모임 (원소가 중복된다면 하나로 간주)  
: 집합은 대문자 알파벳으로 원소는 소문자 알파벳으로 표기함  
: x가 집합 X의 원소라면 x ∈ X로 표현하고 'x는 X에 속한다'라고 읽음  
: x가 집합 X의 원소가 아니라면 x ∉ X 로 표현함

**집합 기호**  

기호 | 이름 | 설명
---|---|---
ℕ | 자연수 집합 | 양의 정수 집합을 나타내며 어떤 경우에는 0을 포함함
ℤ | 정수 집합 | 양의 정수, 음의 정수, 0을 포함하는 집합
ℚ | 유리수 집합 | 분수로 표현 가능한 모든 수의 집합
ℝ | 실수 집합 | 유리수와 무리수를 포함하는 모든 실수의 집합
ℂ | 복소수 집합 | 실수와 허수를 포함하는 모든 복소수의 집합
∅ | 공집합 | 원소가 하나도 없는 집합
∈ | 원소 | 원소가 집합에 포함됨
∉ | 원소가 아님 | 원소가 집합에 포함되지 않음을 의미
⊆ | 부분집합 | 하나의 집합이 다른 집합의 부분으로 포함됨
∪ | 합집합 | 두 집합을 하나의 집합으로 합함
∩ | 교집합 | 두 집합에서 공통된 원소들의 집합
∖ | 차집합 | 한 집합에서 다른 집합의 원소를 뺀 집합
A', Aᶜ | 여집합 | 전체 집합 U에서 집합 A에 속하지 않는 원소들의 집합



## Set Representation
: 집합을 수학적으로 표현하는 방법들  


**원소 나열법**  
: 원소들을 하나씩 나열하는 방식으로 집합을 정의  

```
ex. X = {1, 2, 3}  
```


**조건 제시법**  
: 원소들의 공통된 성질이나 조건을 이용해 집합을 정의  
: 수학적으로 {x | P(x)} 또는 {x : P(x)} 로 표현함  

```
ex. X = { x | 0 < x < 4 인 자연수}  
```



## Set Cardinality
: 집합의 크기 또는 원소 개수와 관련된 개념  


**유한집합**  
: 원소의 개수가 유한하고 이를 명확하게 셀 수 있는 집합  
: 집합 X의 크기는 \|X\| 로 표현함  

```
ex. X = {1, 2, 3} 이라면 집합의 크기는 |X| = 3  
```


**무한집합**  
: 원소의 수가 무한히 많아 개수를 셀 수 없는 집합  

```
ex. ℕ = {1, 2, 3, ...}
ex. ℤ = {...,-1, 0, 1, ...}
```


**공집합**  
: 원소가 하나도 포함되지 않은 집합  
: 원소의 수가 0으로 개수를 셀 수 있으므로 유한집합에 속함  
: ∅ 또는 {} 기호로 표기하며 '파이'라 읽고 수학적으로 \|∅\| = 0 으로 표현함  


**보편 집합**  
: 다른 집합을 모두 포함하는 가장 큰 집합으로 전체 집합이라고 하기도 함  
: 보편 집합은 U로 표시하며 맥락에 따라 Ω, 𝐸, 𝑉 같은 기호가 사용됨  



## Set Relations
: 집합 간 관계를 설명하는 개념들  


**부분집합**  
: 집합 X의 모든 원소가 집합 Y의 원소이면 X는 Y의 부분집합이라 함  
: 수학적으로 X ⊆ Y ⟺ ∀x(x ∈ X → x ∈ Y) 로 표현함  

```
ex. X = {1, 2}, Y = {1, 2} 이면 X ⊆ Y and X = Y
ex. X = {1, 2}, Y = {1, 2, 3} 이면 X ⊆ Y and X ≠ Y
```


**진부분집합**  
: 집합 X가 집합 Y의 부분집합이면서 Y의 원소 개수가 X보다 큰 경우 X를 Y의 진부분집합이라 함  
: 수학적으로 X ⊂ Y ⟺ X ⊆ Y ∧ X ≠ Y 로 표현함  

```
ex. X = {1, 2}, Y = {1, 2, 3} 이면 X ⊆ Y  
```


**상동**  
: 집합 X와 집합 Y의 요소가 동일한 경우 상동이라 함  
: 수학적으로 X ⊆ Y ∧ Y ⊆ X 또는 X = Y 로 표현함  

```
ex. X = {1, 2, 3}, Y = {3, 2, 1} 이면 X = Y  
```


**서로소**  
: 집합 X와 집합 Y의 교집합이 공집합인 경우 서로소라 함  
: 수학적으로 X ∩ Y = ∅ 로 표현함  

```
ex. X = {1, 2}, Y = {3, 4} 이면 X ∩ Y = ∅  
```


**쌍으로 서로소**  
: 여러 집합이 존재하고 임의로 선택한 두 집합의 원소가 서로소면 쌍으로 서로소라 함  

```
ex. X = {1, 2}, Y = {3, 4}, Z = {5, 6} 이면 X, Y, Z는 쌍으로 서로소  
```


**멱집합**  
: 집합 X의 모든 부분집합의 집합을 X의 멱집합이라 함  
: 수학적으로 P(X) 또는 2^X 로 표현함  

```
|P(X)| = 2^|X|

ex. X = {1, 2}의 멱집합은 P(X) = {∅, {1}, {2}, {1, 2}}  
```



## Set Operations
: 집합 간 다양한 연산 방법들로 이를 통해 집합을 조합하거나 교차해 새로운 집합을 만듬  

개념 | 설명
---|---
항등원 | 연산을 적용했을 때 집합의 다른 원소에 영향을 미치지 않는 원소
보편성 | 모든 가능한 원소를 포함하는 전체 집합을 기준으로 하는 성질


**합집합**  
: 두 집합 X와 Y의 합집합은 X 또는 Y에 속하는 원소들의 집합  
: 논리학의 논리합과 개념적으로 동일   

```
X ∪ Y = {x | x ∈ X ∨ x ∈ Y}

ex. X = {1, 2}, Y = {2, 3} 라면 X ∪ Y = {1, 2, 3}  
```


**교집합**  
: 두 집합 X와 Y의 교집합은 X와 Y 모두에 속하는 원소들의 집합   
: 논리학의 논리곱과 개념적으로 동일   

```
X ∩ Y = {x | x ∈ X ∧ x ∈ Y}

ex. X = {1, 2}, Y = {2, 3} 라면 X ∩ Y = {2}  
```


**차집합**  
: 원소가 집합 X에는 속하지만 집합 Y에는 속하지 않는 원소들의 집합  

```
X - Y = {x ∈ U | x ∈ X ∧ x ∉ Y}

ex. X = {1, 2}, Y = {2, 3} 라면 X - Y = {1} 
```


**여집합**  
: 전체집합 U에서 부분집합 X에는 속하지 않는 원소들의 집합  
: 논리학에서 부정과 개념적으로 동일  

```
X' = U - X = {x ∈ U | x ∉ X}

ex. U = {1, 2, 3, 4, 5}, X = {2, 3} 라면 X' = {1, 4, 5} 
```


**대칭하집합**  
: 두 집합 X와 Y의 합집합에는 속하지만 교집합에는 속하지 않는 원소들의 집합  
: 논리학의 배타적 논리합과 부합  

```
X ⊕ Y = {x ∈ U | x ∈ X ∪ Y ∧ x ∉ X ∩ Y}

ex. X = {1, 2, 3}, Y = {2, 3, 4} 라면 X ⊕ Y = {1, 4} 
```


**곱집합**  
: 집합 X의 원소 x와 집합 Y의 원소 y가 순서쌍을 이루며 만들어지는 집합  

```
X × Y = {(x,y)| x ∈ X ∧ y ∈ Y}

ex. X = {1, 2}, Y = {a, b} 라면 X × Y = {(1,a), (1,b), (2,a), (2,b)} 
```



## Set Algebraic Laws
: 집합 연산시 다양한 연산들 간의 관계와 규칙을 정의하는 법칙들


**교환 법칙**  
: 두 집합에 대해 연산 순서를 바꿔도 결과가 변하지 않는 법칙  

연산 | 법칙
---|---
합집합 교환법칙 | X ∪ Y = Y ∪ X
교집합 교환법칙 | X ∩ Y = Y ∩ X
대칭차집합 교환 법칙 | X ⊕ Y = Y ⊕ X


**결합 법칙**  
: 세 집합에 대해 연산의 순서를 바꾸어도 결과가 동일한 법칙  

연산 | 법칙
---|---
합집합 결합법칙 | (X ∪ Y) ∪ Z = X ∪ (Y ∪ Z)
교집합 결합법칙 | (X ∩ Y) ∩ Z = X ∩ (Y ∩ Z)
대칭차집합 결합법칙 | (X ⊕ Y) ⊕ Z = X ⊕ (Y ⊕ Z)


**분배법칙**  
: 하나의 연산이 다른 연산에 대해 분배되는 법칙  

연산 | 법칙
---|---
교집합의 합집합 | X ∩ (Y ∪ Z) = (X ∩ Y) ∪ (X ∩ Z)
합집합의 교집합 | X ∪ (Y ∩ Z) = (X ∪ Y) ∩ (X ∪ Z)


**드모르간 법칙**  
: 합집합과 교집합의 여집합 관계를 나타내는 법칙  

연산 | 법칙
---|---
합집합의 여집합 | (X ∪ Y)' = X' ∩ Y'
교집합의 여집합 | (X ∩ Y)' = X' ∪ Y'

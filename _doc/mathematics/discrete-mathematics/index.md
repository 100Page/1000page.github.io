: 이산 수학은 이산적 데이터를 다루기 위한 수학 분야

**컴퓨터 과학과 이산 수학**  
: 이산 수학은 컴퓨터 과학의 이론적 기반을 제공함  

- 프로그래밍 언어의 논리적 원리에 논리학 사용
- 자료 구조의 이론적 기반 제공
- 알고리즘의 효율성과 복잡성을 분석하는 수학적 도구
- 네트워크 최단 경로, 소셜 네트워크 분석 등에 그래프 이론 사용
- 암호학에 정수론과 조합론이 사용

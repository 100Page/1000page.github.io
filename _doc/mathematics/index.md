# Mathematics
: Mathematics는 그리스어 máthema(배움, 학습)에서 유래  
: 수학은 수, 구조, 공간 등을 연구하는 학문으로 논리적 추론과 증명을 통해 추상적인 개념을 탐구함  
: 컴퓨터 과학에서는 자료 구조, 알고리즘, 그래픽스, 암호학 등에 수학적 원리가 사용됨

## Mathematics Elements
: 수학을 구성하는 기본 개념들과 수학적 사고와 추론의 기반이 되는 요소들  

**기본 요소**

요소| 설명
---|---
수 | 크기, 순서를 나타내는 기본 요소
연산 | 수학적 객체 간 계산과 조작을 위한 규칙과 절차
순서 | 수학적 객체 간 크기나 우선순위를 나타내는 관계
공간 | 점, 선, 면 등을 정의하고 관계를 형성하는 물리적, 추상적 영역


**구조적 요소**

요소| 설명
---|---
집합 | 서로 구별 가능한 수학적 객체들의 모임 
관계 | 수학적 객체들 사이의 연관성이나 상호작용
함수 | 입력과 출력 사이의 규칙적 관계를 나타내는 개념
구조 | 수학적 객체들의 연산과 관계가 이루는 체계 
측도 | 길이, 면적, 부피 등 수학적 객체의 크기를 다루는 개념


**변화적 요소**

요소| 설명
---|---
변화 | 양의 변화를 설명하고 분석하는 개념
위상 | 연속적인 변형에도 불변하는 공간의 구조적 성질을 연구하는 개념
확률 | 불확실성과 무작위성을 수치화하여 분석하는 개념


**사고적 요소**

요소| 설명
---|---
논리 | 수학적 추론과 증명을 통해 올바른 결론을 도출하는 과정
패턴 | 도형, 구조 등에서 규칙성과 반복성 파악하고 이를 일반화하는 과정
추상화 | 핵심 개념을 추출하고 일반화하는 과정
일반화 | 특정 사례나 결과를 더 넓은 범위로 확장하는 과정
형식화 | 개념을 수학적 기호와 정의를 사용해 정확하게 표현하는 과정
대칭성 | 수학적 구조와 관계에서 대칭적 성질을 인식하고 분석하는 과정



## Mathematical Objects
: 수학적으로 정의되고 연산되는 구체적 대상들  
: 특정 성질과 관계를 가지며 수학적 규칙과 연산에 따라 조작 가능한 객체  
: 컴퓨터 과학에서는 컴퓨터 그래픽스, 인공지능 등에서 다차원 데이터를 다루는데 사용  

객체 | 설명
---|---
스칼라 | 크기만 가지고 방향이 없는 단일 수
벡터 | 크기와 방향을 동시에 가진 수의 집합 (1차원 배열)
행렬 | 수를 직사각형 형태로 배열한 2차원 배열 
텐서 | 벡터와 행렬을 일반화한 다차원 배열


**대수적 객체**  
: 대수란 숫자나 기호를 가지고 어떤 규칙에 따라 계산할 수 있는 체계를 의미  

객체 | 설명
---|---
군 (Group) | 하나의 연산에 대해 결합법칙, 항등원, 역원의 성질을 만족하는 집합
환 (Ring)  | 두 개의 연산에 대해 덧셈은 군의 성질을 만족하고 곱셈은 결합법칙을 만족하는 집합
체 (Field) | 두 개의 연산에 대해 모든 원소가 곱셈에 대해 역원을 갖는 집합
모노이드 | 군과 유사하지만 결합법칙과 항등원만 있는 집합
아이디얼 | 환의 부분집합으로 환의 연산을 만족하는 특수한 집합


**기하학적 객체**  
: 공간과 도형의 성질을 연구하는 수학적 객체들

객체 | 설명
---|---
점 | 위치만 있는 공간의 기본 요소
선분 | 두 점을 잇는 직선의 일부
다각형 | 직선으로 이루어진 닫힌 도형
다면체 | 여러 개의 다각형 면으로 이루어진 3차원 도형
원 | 평면 상에서 고정된 점에서 일정한 거리만큼 떨어진 점들의 집합
곡선 | 1차원적 형태를 가지며 연속적인 점들의 집합
곡면 | 2차원적 형태를 가진 공간의 점들의 집합


**해석학적 객체**  
: 연속성과 변화를 다루는 수학적 객체들

객체 | 설명
---|---
수열 | 순서가 있는 수들의 나열
함수 | 각 입력에 대해 정확히 하나의 출력을 가지는 규칙
급수 | 무한히 많은 항을 더하는 형태의 수학적 구조
미분 | 함수의 변화율을 구하는 연산
적분 | 함수 아래의 면적을 구하는 연산
함수공간 | 함수들의 집합으로 이루어진 공간


**위상적 객체**  
: 연속적인 변형 하에서 불변하는 성질을 연구하는 수학적 객체들

객체 | 설명
---|---
위상공간 | 점의 집합과 그 점들 간의 연속적인 관계를 정의한 공간
경로 | 위상공간에서 점들 사이를 연결하는 연속적인 함수
연결성 | 공간 내에서 점들이 하나로 이어져 있는 성질


**조합론적 객체**  
: 이산적인 구조와 그 조합을 다루는 수학적 객체들

객체 | 설명
---|---
순열 | 주어진 객체들로 만들 수 있는 순서 배열
조합 | 순서를 고려하지 않고 주어진 객체들에서 특정 개수를 뽑는 방법
그래프 | 노드(정점)와 엣지(간선)로 구성된 수학적 구조
트리 | 노드들이 간선으로 연결된 비순환 그래프
파티션 | 집합을 여러 개의 부분집합으로 나누는 방법
매트로이드 | 집합의 부분집합에 대한 특정 조건을 만족하는 조합론적 구조


**확률적 객체**  
: 불확실성과 무작위성에 관한 수학적 객체들

객체 | 설명
---|---
확률공간 | 확률이 정의된 집합과 관련된 구조
확률변수 | 확률공간에서 정의되고 실수 값을 가지는 변수
확률분포 | 확률변수의 모든 가능한 값과 그에 대한 확률을 나타내는 함수
확률과정 | 시간에 따라 변화하는 확률변수들의 집합
샘플공간 | 실험에서 가능한 모든 결과를 포함하는 집합



## Mathematical Methods
: 추상화, 형식화, 일반화 개념을 통해 문제를 수학적으로 표현하고 분석하는 방법론  
: 방법론은 독립적으로 사용 가능하며 복잡한 문제인 경우 상호 보완적으로 사용됨  


**모형화**  
: 현실 세계의 문제나 현상을 수학적으로 표현하고 분석하는 과정  

```
문제 정의 => 가정 설정 => 수학적 모델 구축 => 모델 분석 및 해결 => 결과 해석 및 검증
```

모형 종류 | 설명
---|---
결정론적 모형 | 입력 값이 같으면 항상 동일한 출력 값이 나오는 모형으로 무작위성과 불확실성이 없음
확률적 모형 | 동일한 입력에도 다양한 출력 값이 나오는 모형으로 무작위성과 불확실성을 포함


**공리론**
: 기본 가정(공리)에서 출발해 논리적 추론을 통해 수학 체계를 구축

```
기본 개념 정의 => 공리 설정 => 정리 도출 => 체계 구축
```


**증명**  
: 수학적 명제의 진위를 논리적으로 입증하는 과정  

```
명제 이해 => 증명 전략 선택 => 논리적 추론 => 결론 도출
```

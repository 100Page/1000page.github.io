# Glue Language
: 서로 다른 소프트웨어 구성 요소를 연결하고 통합하는데 사용되는 프로그래밍 언어  
: 접착 언어들은 공통적으로 동적 타이핑 언어와 인터프리터 언어의 특징을 가짐  
: 시스템 간의 데이터 변환, 파이프라인 구축, 자동화에 특화됨  

**사용 사례**
- 시스템 관리 및 자동화
- 데이터 변환 및 처리
- API 통합
- 테스트 자동화
- 로그 분석


**언어 종류**
- Perl
- Ruby
- Python
- JavaScript
- Shell Script

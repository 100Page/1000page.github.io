# Contract-Oriented Programming Language
= Smart Contract Programming Language   
: 계약 지향 프로그래밍 언어  
: 스마트 계약을 구현하기 위한 프로그래밍 언어  
: 소프트웨어 모듈 간 상호 작용을 계약이라는 개념을 통해 정의

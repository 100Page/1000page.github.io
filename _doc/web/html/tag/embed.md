# HTML Embed Tag
: 외부 리소스를 삽입하기 위한 태그  


속성 | 설명
---|---
src  | 외부 리소스 url 지정
type | 외부 리소스 MIME 타입 명시  
width, height | 외부 리소스의 너비와 높이 지정


```html
<embed src="example.pdf" type="application/pdf">

<embed src="video.mp4" type="video/mp4" width="640" height="480">
```

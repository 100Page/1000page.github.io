**보안이 실패하는 이유**  
- 기술적 요인
  - 시스템 복잡성 증가
  - 기술 변화 속도 : 기술 발전으로 기존 보안 대책이 무력화
  - 레거시 시스템 문제
- 관리적 요인
  - 보안 인식 부족
  - 예산이나 자원 제약
- 인적 요인
  - 사용자 실수
  - 보안 정책 미준수
  - 내부자 위협
- 환경적 요인
  - 새로운 위협의 등장
  - 공격 기술의 진화
  - 규제 환경 변화
